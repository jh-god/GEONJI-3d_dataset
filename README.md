# GEONJI-3d: A Multimodal Dataset for 3D Perception in Unstructured Outdoor Environments

GEONJI-3D is a multimodal dataset collected mainly in the Geonji Mountain Research Forest, and provides sensor data from LiDAR, cameras, and IMUs in the ROS 2 bag format.

## env
- Ubuntu 22.04
- ROS2 Humble
